# Совмещенный ABC-XYZ анализ товарного ассортимента интернет-магазина "Ultra-Fresh"

## 📌 Описание проекта
Проект посвящен оптимизации складских запасов и повышению эффективности коммерческого отдела интернет-магазина свежих овощей и грибов (категория Ultra-Fresh). На основе исторического массива транзакций (более 470 тыс. кг проданной продукции) был проведен комплексный анализ ассортимента по двум ключевым метрикам: объему генерируемой выручки (ABC) и стабильности спроса по месяцам (XYZ).

**Цель проекта:** Выявить ключевые драйверы прибыли, локализовать зоны риска с высокой вероятностью списаний из-за порчи товаров и сформировать стратегические рекомендации для категорийного менеджмента.

---

## 🛠️ Технический стек и этапы предобработки
* **Инструменты:** MS Excel, Сводные таблицы (Pivot Tables), Сводные диаграммы.
* **Предобработка данных (Data Cleaning):** 
  В ходе анализа была обнаружена и устранена критическая системная ошибка в исходном датасете: в столбцах количества товаров часть данных была ошибочно конвертирована системой в формат дат (например, вес `11.05` превратился в `11.май`). Была проведена ручная и формульная очистка, данные приведены к единому числовому формату для корректного расчета сумм.
* **Стабилизация вычислений:** При расчете финальной матрицы формульные значения `СУММЕСЛИ` были зафиксированы как жесткие значения во избежание циклического пересчета и хаотичного сдвига строк при сортировке.

## 📈 Методология анализа
1. **ABC-анализ (по Выручке в $):** Категории отсортированы по убыванию вклада в общий оборот ($3 369 766,48). Границы классов: А (до 70%), В (70-95%), С (95-100%).
2. **XYZ-анализ (по стабильности спроса в кг):** Продажи сгруппированы помесячно за весь период. Для каждой категории рассчитан **Коэффициент вариации ($CV$)**:
   $$\text{CV} = \frac{\text{Стандартное отклонение}}{\text{Среднее значение}} \times 100\%$$
   Границы классов: X (до 10% — спрос стабилен), Y (10-25% — умеренные колебания), Z (более 25% — хаотичный/сезонный спрос).

## 💡 Главные бизнес-инсайты и рекомендации

### 1. Группа AY (Локомотивы бизнеса) — `Flower/Leaf B Vegetables`, `Cabbage`
* **Инсайт:** Всего 2 категории обеспечивают **54,4% всей выручки** компании. Спрос имеет предсказуемые сезонные колебания.
* **Решение:** Дефицит этих позиций критичен. Требуется жесткий контроль остатков и поддержание **страхового запаса в размере 15–20%** от среднемесячных продаж для сглаживания пиков.

### 2. Группа BZ (Зона повышенного риска) — `Aquatic Tuberous Vegetables`, `Solanum`
* **Инсайт:** Категории приносят высокий доход (~30%), но спрос критически нестабилен. Коэффициент вариации водных овощей составляет **48,9%** (жесткий пик потребления осенью). В межсезонье закупки по старым нормативам ведут к масштабным списаниям скоропортящейся продукции Ultra-Fresh.
* **Решение:** Внедрить **сезонное квантование закупок** (динамическое изменение объема заказа в зависимости от месяца). Перейти на долгосрочные гибкие контракты с поставщиками для работы по системе *Just-in-Time* (поставки «день-в-день»).

### 3. Группа CZ (Кандидаты на оптимизацию) — `Edible Mushroom`
* **Инсайт:** Минимальный вклад в выручку (5,67%) в сочетании с высокой хаотичностью спроса (32,9%). Короткий срок жизни грибов делает эту категорию главным источником чистых убытков на складе.
* **Решение:** Сократить ширину ассортимента внутри категории (оставить только топ-1 ходовую позицию). Уменьшить полочное пространство в торговом зале, закупать строго минимальными квантами.
